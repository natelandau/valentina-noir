## Options and Settings

Administrators are able to configure the following options and settings for a campaign:

-   **Experience** - Who is able to grant experience to players. This can be set to:
    -   **Unrestricted** - Any user can grant experience to players.
    -   **Player** - Only players can grant experience to players.
    -   **Storyteller** - Only the storyteller can grant experience to players.
-   **Campaign Management** - Who is able to manage the campaign and create, update, and delete campaigns, books, and chapters.
    -   **Unrestricted** - Any user can manage the campaign.
    -   **Storyteller** - Only the storyteller can manage the campaign.
-   **Trait Management** - Who is able to update traits and their values for characters in the campaign _without needing to spend experience points_. This can be set to:
    -   **Unrestricted** - Any player can change a trait's value on any character at any time bypassing the experience point cost.
    -   **Within 24 Hours** - The player who owns the character can change a trait's value for free within the last 24 hours of the character being created.
    -   **Character Owner** - The owner of the character can change a trait's value bypassing the experience point cost.
    -   **Storyteller** - Only the storyteller can change a trait's value bypassing the experience point cost.
-   **XP Cost for Character Autogen** - The experience spent to autogenerate character(s)
-   **Number of Characters to Autogen** - Should users be able to autogenerate multiple characters and choose the one they want to keep? If so, how many?
