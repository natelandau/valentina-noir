# Character Concepts

Character concepts grant specific specialties and abilities that define a character's role and playstyle.

## Assigning Concepts

The autogeneration engine automatically assigns concepts to new characters.

When creating characters manually, you can optionally assign a concept during creation.

## Built-in Character Concepts

### Ascetic

Disciplined martial artists harness inner chi to perform incredible feats and attacks.

**Specialties:**

| Name | Description |
| --- | --- |
| **Focus** | Gather chi to resist gases, poisons, and psionic attacks. Hold breath for one turn per existing `stamina` + `willpower`. Immune to the vampiric discipline `Dominate`. |
| **Iron Hand** | Deliver a single punch per scene. Augment damage by spending `willpower` (1 point per damage level). |

### Berserker

Fierce warriors tap into primal rage to gain incredible strength and combat prowess.

**Specialties:**

| Name | Description |
| --- | --- |
| **Frenzy** | Ignore the first inflicted health levels of damage with no penalty up until `Mauled`. Cannot be stunned. Gain an automatic success on any strength roll once per turn (equal to a permanent dot in Potence). |

### Businessman

Astute individuals excel at commerce and negotiation, skilled in deal-making and resource management.

**Specialties:**

| Name | Description |
| --- | --- |
| **Persuasion** | Win over enemies with facts and logic. This sustained power enthralls targets through rational argument. |
| **Wealth** | Access significant resources and generate more wealth. Start with `Resources` of `2`. |

### Crusader

Dedicated sentinels sworn to a code of conduct, armed with divine, academic, and martial skills.

**Specialties:**

| Name | Description |
| --- | --- |
| **Incorruptible** | Heal 1 health level to a target once per turn. Automatically gain one success on any First Aid or medicine roll. Start with `Faith` of `1` (equivalent to a Discipline). Repel supernatural beings for every success on a Faith roll. |
| **Weapon of Choice** | Choose a single weapon and remain loyal to it. Re-roll any single roll with this weapon once per turn. Gain a new specialization at `3`, `4`, and `5` dots, granting an additional die when using the specialized weapon. |

### Healer

Devout servants of gods or higher powers possess the ability to heal and protect.

**Specialties (Choose 1):**

| Name | Description |
| --- | --- |
| **Heal** | Heal 3 health levels to a target once per turn. Automatically gain one success on any First Aid or medicine roll. |
| **True Faith** | Start with `Faith` of `3` (equivalent to a Discipline). Repel supernatural beings for every success on a Faith roll. |

### Performer

Charismatic performers and spellcasters use artistry and magic to inspire and manipulate.

**Specialties:**

| Name | Description |
| --- | --- |
| **Fast Talk** | Automatically succeed on any `charisma`, `expression`, or `performance` roll. Command immediate attention, freezing enemies (including groups) for the first turn of combat. Must continue performing to maintain the effect. |

### Scientist

Experts draw power from esoteric knowledge, wielding unique and potent abilities through specialized gear.

**Specialties:**

| Name | Description |
| --- | --- |
| **Delicate Equipment** | Choose any Thaumaturgical or Blood Magic Path from the Vampire or Sorcerer book. Apply `3` dots distributed as you prefer. Represent these as tools or scientific equipment that generate the effect (e.g., Lure of Flames as a flamethrower, Lightning as weather equipment). Equipment must be present to create the effect. |

### Shaman

Nature-focused spiritualists wield the power of the natural world.

**Specialties (Choose 2):**

| Name | Description |
| --- | --- |
| **Familiar** | Gain a trained animal that carries out simple commands. |
| **Friend of Animals** | Function as `Animalism` level `1`. |
| **Spirit Sight** | See the dead, naturalistic spirits, and places of power. Detect supernatural beings and penetrate spells, illusions, and glamour on a `perception` + `occult` roll (difficulty `4`). |
| **Read Auras** | Learn various qualities of a person from their aura. |
| **Astral Projection** | Free your mind to travel the world in astral form. |
| **Remove Frenzy** | Cool vampire frenzy, werewolf rage, and similar states. Target becomes passive for two turns. |

### Soldier

Skilled warriors possess a wide range of combat abilities and weapon expertise.

**Specialties (Choose 1):**

| Name | Description |
| --- | --- |
| **Firearms Specialist** | Re-roll any single Firearms roll once per turn. Specialize in new firearms at `3`, `4`, and `5` dots (grants an additional die when using specialized weapons). |
| **Hand-to-hand Specialist** | Re-roll any single `Brawl` roll once per turn. Gain new specializations at `3`, `4`, and `5` dots (grants an additional die when using specialized martial arts styles). |
| **Melee Specialist** | Re-roll any single `Melee` roll once per turn. Gain new specializations at `3`, `4`, and `5` dots (grants an additional die when using specialized melee weapons). |

### Tracker

Skilled hunters and trackers maintain a deep connection to wilderness or urban environments.

**Specialties:**

| Name | Description |
| --- | --- |
| **Camouflage** | Camouflage into your preferred environment given 1 turn of preparation. Can be detected on a `Perception` or `Focus` roll (difficulty `8`). Attacks from this position count as surprise attacks. |
| **Surprise Attack** | Deal an additional `3` successes of damage on surprise attacks. Subsequent attacks no longer count as surprise unless plausibly silent. |

### Tradesman

Skilled artisans and laborers excel in specific trades or crafts (blacksmithing, carpentry, alchemy, etc.).

**Specialties:**

| Name | Description |
| --- | --- |
| **Hardiness** | Function as `Fortitude` `1`. Automatically soak `1` success on all attacks at no cost. |
| **Handy** | Gain a free dot in `Repair` and `Crafts`. |

### Under-worlder

Sneaky and dexterous individuals excel in stealth, lock picking, and traps.

**Specialties:**

| Name | Description |
| --- | --- |
| **Tools of the Trade** | Possess specialized equipment (lockpicks, laser drill, getaway car, printing press) that decreases difficulty by `2` when used. |
| **Professional** | Perform any single `security` roll at `-1` difficulty. |
| **Lay Low** | Leave no paper trail. Maintain multiple alternative identities with documents that withstand all but sustained investigation. Gain the background `Arcane` at `2` dots. |
